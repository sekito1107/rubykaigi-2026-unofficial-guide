# RubyKaigi 2026 Fan Guide

> [!IMPORTANT]
> **これは RubyKaigi 2026 の非公式ファンガイドです。**
> 
> 公式のスケジュールや正確な情報は必ず [公式サイト](https://rubykaigi.org/2026/) をご確認ください。

RubyKaigi 2026 をより楽しむための、初心者向けの非公式ガイドアプリケーションです。
React + TypeScript + Vite + Tailwind CSS v4 で構築されています。

## 主な機能 (予定)
- 📅 3日間のスケジュール一覧表示
- 🔍 セッション検索機能
- ⭐ お気に入りセッション登録

## 開発の始め方

```bash
# 依存関係のインストール
npm install

# 開発用サーバーの起動
npm run dev
```

## 技術スタック
- **Frontend**: React 19, TypeScript
- **Styling**: Tailwind CSS v4
- **Build Tool**: Vite
